Projects

TankWars

TankWars was the final assignment for a year one computer graphics paper (CGRA151). We had a lot of freedom to create any 2-D game or digital artwork we desired. TankWars is an imitation of a popular online game called Tank Trouble with slightly different weapons and feeling.

Auckland Mapper*

This was the 1st assignment for a year 2 algorithm and data structure course (COMP261). It features many different algorithms and data structures including the following; A*, Articulation point identifier, Directed graph, Trie & a Quadtree.

3-D Renderer*

This was the 3rd assignment for a year 2 algorithm and data structure course (COMP261). It uses a Z-buffer rendering pipeline to display different 3-D objects with directional Gouraud shading to allow surfaces to appear curved, it can also handle multiple light sources.
